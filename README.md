### Get live, real-time updates on the prices of your favorite cryptocurrencies with intuitive and user-friendly app.

* Xcode 16, minimum iOS version 15.
* [Binance API](https://developers.binance.com/docs/binance-spot-api-docs/testnet/rest-api/general-endpoints) and [Coingecko API](https://docs.coingecko.com/v3.0.1/reference/coins-markets).
* You can use the existing test API key from this repo. Coingecko limitation: 30 requests/min.
* Clean Architecture. Many dependencies are included in this test task to demonstrate proper project structuring.
* Simple navigation logic is used to avoid spending unnecessary time on the test task.
* Includes two unit tests to showcase my approach while keeping the test task time-efficient.
* UI tests added.
